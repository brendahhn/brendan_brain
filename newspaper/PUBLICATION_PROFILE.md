# Publication Profile
Governing policy: system/PUBLICATION_POLICY.md (section order, budgets, checklist).
Delivery: edition file in newspaper/editions/YYYY-MM-DD.md (durable channel) + push
notification with 3-5 headlines. Gmail draft optional/best-effort (known flaky).
Edition frontmatter records: sections included, word counts, items cut and why (one line
each), publisher verdict.
Annotation intake: Brendan edits the edition file inline (⭐ 🙂 ❌ / freeform lines starting
with `>>`) or drops a file in newspaper/annotations/. Processing: brain-annotations skill.
