# Communication Preferences (evidence-based; from robot history)

- **Lead with what changed / what matters.** All robots use headline-first formats.
- **No repetition.** FootyBot v3.2 recalibration exists because output got "stale, lazy,
  repetitive." Coverage ledgers are mandatory for recurring surfaces.
- **Confidence tiers on claims** (S/A/B/C/Speculative pattern) — carry into the newspaper.
- **Don't reward length.** Empty sections say so in one line or vanish.
- **Questions must be answerable.** Include context so Brendan can answer without opening
  files ("Decisions Needed" pattern from Trading/Health robots).
- **Gmail drafts are best-effort**, repo files are the durable channel; push notifications
  carry 3-5 headlines max.
- Emoji as section markers is established robot style (💼 ⚡ 🩺 ❓) — fine to keep, don't overdo.
